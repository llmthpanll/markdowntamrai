### Computer Type ประเภทของคอมพิวเตอร์

เราเเบ่งประเภทของคอมพิวเตอร์ได้ตามขนาดเเละความสามารถในการประมวลของคอมพิวเตอร์ออกเป็น 4 ประเภทใหญ่ ดังนี้ Microcomputers (Personal Computers) / Minicomputers (Mid-range computers) / Mainframe Computers เเละ Supercomputers

![enter image description here](https://github.com/misterfocusth/ToBeIT-67-ITF/blob/main/Images/6.png?raw=true)

- **Microcomputers (Personal Computers)** ได้เเก่ Personal Computer / Mobile Phone / Game Console เเละ Embedded Computer

- **Minicomputers (Mid-range Computers)** คอมพิวเตอร์กลางที่มีขนาดใหญ่ขึ้นมาจาก Personal Computer เหมาะกับการใช้ในการประมวลผลข้อมูลจำนวนเยอะๆ เก็บข้อมูลผู้ใช้จำนวนมากๆ เเละการทำงานเเบบ Multitasking

- **Mainframe Computer** คอมพิวเตอร์ขนาดใหญ่ที่ใช้ในองค์กรเพื่อประมวลผลข้อมูลจำนวนมาก

- **Supercomputers** เป็นคอมพิวเตอร์ที่มีความสามารถสูงที่สุดในกลุ่มมีขนาดใหญ่ ซูเปอร์คอมพิวเตอร์ สามารถประมวลผลข้อมูลในปริมาณมากรวมถึงการประมวลผลงานที่มีรูปแบบอันซับซ้อน มีความรวดเร็วในการคำนวณได้มากกว่าหนึ่งล้านล้านต่อวินาที (1 Trillion calculations per second)

### Computer Hardware Components อุปกรณ์คอมพิวเตอร์

คอมพิวเตอร์จะสามารถทำงานได้ก็ต้องมีฮาร์ดเเวร์หรืออุปกรณ์ภายในคอมพิวเตอร์ในการขับเคลื่อนให้คอมพิวเตอร์สามารถทำงานได้ เเละประมวลผลข้อมูลในรูปเเบบต่างๆ ในบทนี้น้องก็ได้เรียนรู้เกี่ยวกับฮาร์ดเเวร์ หรืออุปกรณ์ภายในคอมพิวเตอร์ของเรา ไม่ว่าจะเป็น

![enter image description here](https://github.com/misterfocusth/ToBeIT-67-ITF/blob/main/Images/13.png?raw=true)

- **Motherboard** : แผงวงจรหลักของคอมพิวเตอร์ที่รวมอุปกรณ์ต่าง ๆ ไว้ด้วยกัน ไม่ว่าจะเป็นซีพียู แรม การ์ดจอ ฮาร์ดดิสก์ และอื่น ๆ พร้อมทั้งทำหน้าที่ควบคุมและสั่งการอุปกรณ์เหล่านี้ให้ทำงานตามคำสั่ง โดยนอกจากมันจะถูกเรียกว่าเมนบอร์ดแล้ว ก็ยังมีชื่อเรียกอื่นอีกหลายอย่าง เช่น มาเธอร์บอร์ด (Motherboard), ซิสเต็มบอร์ด (System Board) หรือลอจิคบอร์ด (Logic Board)

- **Processor** : หน่วยประมวลผลกลาง (Central Processing Unit) หรือ CPU (ซีพียู) หรือ โปรเซสเซอร์ (Processor) เป็นองค์ประกอบที่สำคัญส่วนหนึ่งของเครื่องคอมพิวเตอร์ทุกประเภทในโลก เปรียบเสมือนกับเป็นมันสมองให้กับคอมพิวเตอร์นั่นเอง ประกอบไปด้วยวงจรทางไฟฟ้ามากมาย ที่อยู่บนแผ่นซิลิกอนซิป  ซึ่งมีขนาดเล็กมาก ๆ

- **Cooling Devices** : ระบบระบายความร้อนของ CPU ( CPU Cooling System ) ในเครื่อง PC นับเป็นอีกหนึ่งอุปกรณ์ที่มีความสำคัญมากต่อคอมพิวเตอร์เนื่องมาจากการทำงานของ CPU นั้นย่อมทำให้เกิดความร้อนขึ้น เมื่อมีความร้อนเกิดขึ้นแต่ไม่มีการระบายความร้อนก็จะทำให้ประสิทธิภาพในการทำงานลดลงและจะตามมาด้วยอาการที่คุ้นเคยเช่น คอมรวม ค้าง หรือ อาการผิดปกติของคอมพิวเตอร์อื่นๆนั่นเอง ดังนั้นจึงต้องมีตัวช่วยในการระบายความร้อนของ CPU นั่นเอง

- **Memory** : ใช้ในการเก็บข้อมูลที่คอมพิวเตอร์ต้องใช้ในการประมวลผล ไม่ว่าจะเป็นการเก็บข้อมูลเเบบชั่วคราว เเละเเบบถาวร โดย Memory จะถูกเเบ่งออกได้ 2 ประเภท ได้เเก่ Volatile Memory คือหน่วยเก็บข้อมูลเเบบชั่วคราว ข้อมูลจะหายทันทีถ้าไม่มีไฟฟ้า หรือเราปิดอุปกรณ์คอมพิวเตอร์ เช่นการเก็บข้อมูลภายใน RAM เเละ CPU L1/ L2/ L3 caches เเละ Non-Volatile Memory คือหน่วยเก็บข้อมูลเเบบถาวร ข้อมูลจะไม่หายถึงเเม้ว่าเราจะปิดเครื่อง

- **Adapter** : Peripheral Component Interconnection คือช่องเสียบอุปกรณ์ ต่าง ๆ อาทิ การ์ดเสียง การ์ด Network โมเด็มแบบ Internal แม้กระทั่งการ์จอ เพื่อให้อุปกรณ์ทำงานควบคู่ไปกับซีพียูได้อย่างรวดเร็ว ซึ่ง PCI มีความสามารถที่จะส่งข้อมูลด้วยความเร็วสูง

- **Power Supply** : อุปกรณ์คอมพิวเตอร์ที่ทำหน้าที่แปลงไฟฟ้ากระแสสลับให้เป็นกระแสตรงเพื่อจ่าย ให้กับอุปกรณ์อื่น เช่น เมนบอร์ด ซีพียู แรม เเละฮาร์ดดิสก์

- **Storage** : ชิ้นส่วนของฮาร์ดแวร์คอมพิวเตอร์ที่ใช้สำหรับบันทึกพกพาและดึงข้อมูลออกมา สามารถเก็บและรักษาข้อมูลในระยะสั้นหรือระยะยาว

### Input Output Devices อุปกรณ์นำเข้าเเละส่งออกข้อมูล

คอมพิวเตอร์จะสามารถทำงานได้ก็ต่อเมื่อเรามีอุปกรณ์ที่นำเข้าข้อมูล ไปให้คอมพิวเตอร์เพื่อใช้ในการประมวลผล จากนั้นคอมพิวเตอร์ก็จะทำการส่งออกข้อมูลหรือเเสดงผลข้อมูลออกมาให้เรา ในบทนี้เราจะมาดูกันว่าอุปกรณ์ที่ใช้ในการนำเข้าเเละส่งออกข้อมูลมีอะไรบ้างงงง !

![enter image description here](https://github.com/misterfocusth/ToBeIT-67-ITF/blob/main/Images/25.png?raw=true)

- **Motion Input** : อุปกรณ์ในการนำเข้าข้อมูลท่าทางการเคลื่อนไหวเเละขยับร่างกาย

- **Voice Input** : อุปกรณ์ในการนำเข้าข้อมูลเสียง เช่นไมโครโฟน เป็นต้น

- **Video Input** : อุปกรณ์ในการนำเข้าข้อมูลวิดีโอ เช่น กล้องถ่ายรูป กล้องเว็บแคม เป็นต้น

- **Reading Devices** : อุปกรณ์ในการสเเกนข้อมูล หรือเอกสาร เช่น เครื่องสเเกนเนอร์ (Scanner) เป็นต้น

![enter image description here](https://github.com/misterfocusth/ToBeIT-67-ITF/blob/main/Images/26.png?raw=true)

- **Keyboard** : เป็นอุปกรณ์หลักที่ใช้ในการนำข้อมูลลงในเครื่องคอมพิวเตอร์ผ่านการพิมพ์เเละสัมผัสลงบนเเป้นคีย์บอร์ด

- **Pointing Devices** : เป็นอุปกรณ์ที่ใช้ชี้ตำแหน่งการทำงานรวมถึงสั่งการให้คอมพิวเตอร์ทำงานบางคำสั่งที่มีการโต้ตอบกันระหว่างผู้ใช้กับคอมพิวเตอร์ โดยใช้มือเป็นตัวบังคับทิศทางและใช้นิ้วสำหรับการกดเลือกคำสั่งงาน

- **Touch Screen** : จอภาพที่รับข้อมูลได้จากการสัมผัสที่ผิวจอ เป็นต้นว่า บนจอแสดงหัวข้อให้เลือก เราสามารถเลือกได้โดยใช้นิ้วจิ้มลงไปที่ตำแหน่งที่ต้องการ เป็นการแสดงว่า คอมพิวเตอร์ได้รับข้อมูลแล้ว

- **Pen Input** : อุปกรณ์สำหรับนำเข้าข้อมูลโดยการสัมผัสไปที่หน้าจอผ่านหัวปากกา เช่น Stylus Pen เเละ Apple Pencil เป็นต้น

![enter image description here](https://github.com/misterfocusth/ToBeIT-67-ITF/blob/main/Images/27.png?raw=true)

- **Projector** : อุปกรณ์ฉายภาพที่ใช้ในการนำเสนอข้อมูลต่างๆ โดยสามารถรองรับสัญญาณภาพจากคอมพิวเตอร์, เครื่องเล่นวีซีดี, เครื่องเล่นดีวีดี, และเครื่องกำเนิดภาพอื่นๆ

- **Printer** : อุปกรณ์เครื่องพิมพ์ที่รับสัญญาณตรงจากเครื่องคอมพิวเตอร์เพื่อพิมพ์งานออกมาเป็นข้อความ ภาพ ลงบนกระดาษหรือวัตถุอื่นในประเภทเดียวกัน

- **Display / Monitor** : จอแสดงผลสำหรับคอมพิวเตอร์ หรืออุปกรณ์อิเล็กทรอนิกส์แบบอื่นๆ ที่มีจอแสดงภาพ

- **Audio Output** : อุปกรณ์ในการส่งออกข้อมูลเสียง เช่น ลำโพง เป็นต้น

### Software Type ประเภทของซอฟต์เเวร์

น้องๆ รู้มั้นว่าซอฟต์เเวร์ที่เราใช้กันในปัจจุบันเนี่ย มีเยอะเเล้วก็หลากหลายประเภทเลย เเต่จริงๆ เเล้วเราสามารถเเบ่งประเภทของซอฟต์เเวร์ได้ตามประเภทเเละการใช้งาน ได้ตามนี้เลย

- **Retail Software** : ซอฟต์เเวร์ที่พัฒนาขึ้นเเละมีการจัดจำหน่าย หากต้องหารใช้งานจะต้องซื้อลิขสิทธิ์มาก่อนจิงจะสามารถใช้งานได้

- **Custom Software** : ซอฟต์เเวร์ที่ทำขึ้นตามความต้องการของบุคคลหรือบริษัท โดยอาจจัดทำขึ้นเพื่อใช้เป็นการภายใน หรือจุดประสงค์พิเศษ

- **Web App** : เว็บเเอปพลิเคชั่น ผู้ใช้สามารถเข้าใช้งานได้ทันทีผ่านตัวเว็บบราวเซอร์

- **Mobile App** : เเอพพลิเคชั่นบนโทรศัพท์มือถือ เป็นเเอพที่พัฒนาให้เหมาะกับการใช้งานบนโทรศัพท์มือถือ เเละส่วนมากต้องมีการดาวน์โหลด เเละติดตั้งก่อนจึงจะสามารถใช้งานได้

- **Shareware** : ชนิดของโปรแกรมที่ผู้เป็นเจ้าของแจกจ่ายให้ผู้ใช้โดยไม่ต้องเสียค่าใช้จ่ายใดๆ ในลักษณะของการทดลองใช้งาน และมักมีการจำกัดความสามารถของโปรแกรมที่ใช้งานได้

- **Freeware** : ซอฟต์แวร์ที่สร้างขึ้นและสามารถนำไปใช้ได้ในทุกจุดประสงค์โดยไม่ต้องเสียค่าใช้จ่าย

- **Open Source Software** : เป็นหลักการของซอฟต์แวร์นั้นให้บุคคลภายนอก ผู้ใช้งาน ผู้พัฒนารายอื่นได้ใช้งาน รวมถึงการนำ รหัสต้นฉบับ หรือ ซอร์สโค้ด (Source Code) ของซอฟต์แวร์ไปแจกจ่าย ทำซ้ำ ศึกษา ดัดแปลง พัฒนาต่อยอดได้อีก โดยไม่ผิดกฏหมาย หรือกฏการใช้แต่อย่างใด

- **Public Domain Software** : ซอฟต์แวร์ที่สร้างขึ้นแล้วผู้สร้างหรือเจ้าของยินยอมให้ใคร ๆ มาคัดลอกนำไปใช้ได้ โดยไม่ผิดพระราชบัญญัติลิขสิทธิ์ เช่น โปรแกรม Cuwriter ในระบบเครือข่าย จะมีกลุ่มโปรแกรมกลุ่มหนึ่งเรียกว่า PDS ย่อมาจาก public domain software เป็นที่รวมของโปรแกรมฟรีต่าง ๆ ที่ใคร ๆ ก็สามารถคัดลอกไปใช้ได้

![enter image description here](https://github.com/misterfocusth/ToBeIT-67-ITF/blob/main/Images/55.png?raw=true)

Software Categories - หมวดหมู่ของซอฟต์เเวร์

- **Productivity (Business & Personal)** : ซอฟต์เเวร์เพื่อการทำงาน หรือการใช้งานส่วนบุคคล

- **Graphic and Media** : ซอฟต์เเวร์เพื่อใช้ในการออกเเบบ หรือพัฒนาด้านกราฟฟิกเเละมัลติมีเดีย

- **Personal Interest** : ซอฟต์เเวร์ที่มีการปรับเปลี่ยนเนื้อหาให้เหมาะกับสิ่งที่ผู้ใช้สนใจ หรือมีการใช้งานบ่อยๆ

- **Communication** : ซอฟต์เเวร์เพื่อการติดต่อสื่อสาร

- **Security** : ซอฟต์เเวร์เพื่อการรักษาความปลอดภัย

- **File, Disk and System Management** : ซอฟต์เเวร์เพื่อการจัดการไฟล์ ดิส หรือไฟล์ของระบบปฎิบัติการ

### Operating System
 -   OS คือตัวกลางในการประสานงานการทำงานระหว่างฮาร์ดแวร์กับซอฟต์แวร์ เพื่อตอบสนองให้ผู้ใช้งานให้เป็นประโยชน์และมีประสิทธิภาพสูงสุด
 - วิธีการติดต่อกับผู้ใช้
	 - คอมมานด์ไลน์ (CLI : Command Line Interface)
	 - กราฟิก (GUI : Graphical User Interface)
 - หน้าที่สำคัญของ OS
	 - การจัดการเกี่ยวกับไฟล์
	 - การจัดการเกี่ยวกับอุปกรณ์ input/output
	 - การจัดการหน่วยความจำ
	 - การจัดการเกี่ยวกับโปรแกรมประยุกต์
	 - การจัดลำดับความสำคัญก่อนหลังในการทำงาน

### Intro to Database
 -  **Data** คือข้อมูลดิบที่ยังไม่ผ่านการประมวลผล เช่น คะแนนสอบ
 -  Information คือข้อมูลที่ผ่านการประมวลผลแล้ว เช่น เกรด
 - **Database** คือ กลุ่มข้อมูลขนาดใหญ่ที่ถูกเก็บรวบรวมไว้ที่ใดที่หนึ่ง โดยเป็นข้อมูลที่มีความสัมพันธ์กัน ซึ่งถูกจัดเก็บอย่างเป็นระบบ โดยมีซอฟต์แวร์เข้ามาควบคุมกระบวนการใช้งาน การทำงาน หรือการประมวลผล ทำให้ผู้ใช้สามารถใช้ข้อมูลได้อย่างมีประสิทธิภาพ

### Intro to Network & Internet
 - **Network** คืออุปกรณ์ 2 อย่างขึ้นไปทำการเชื่อมต่อกัน โดยสามารถติดต่อสื่อสารกันและแลกเปลี่ยนข้อมูลกันได้ เช่น โทรศัพท์ 2 เครื่อง ส่งข้อมูลหากันได้
 - **Internet** คือ Network of Networks การนำเครือข่ายย่อย ๆ หลาย ๆ เครือข่ายมาเชื่อมต่อกัน เพื่อทำให้เกิดการติดต่อสื่อสารขนาดใหญ่ขึ้น
 - **Network Topology** คือ คือ รูปแบบการเชื่อมต่อเครือข่ายในแบบต่างๆ ที่ใช้เพื่อรับส่งข้อมูลระหว่างอุปกรณ์อิเล็กทรอนิกส์ ตั้งแต่ 2 เครื่องเป็นต้นไป ซึ่งเป็นพื้นฐานสำคัญในการสร้างและออกแบบ Network
	- **Bus Topology**
		![enter image description here](https://cdn.discordapp.com/attachments/974043170037956649/1032333453909315734/unknown.png)
		- ทุกอุปกรณ์ต่อเข้ากับสายสื่อสารหลักเพียงสายเดียวหรือที่เรียกว่า Backbone หรือ Bus ซึ่งการส่งข้อมูลสามารถส่งได้เพียงครั้งละเครื่องเท่านั้น เมื่อมีการส่งต่อข้อมูล อุปกรณ์ทุกเครื่องจะตรวจสอบว่ารับข้อมูลได้หรือไม่ ถ้าใช่ก็รับมา ถ้าไม่ใช่ก็ปล่อยข้อมูลไป โดยเป็นการเชื่อมต่อที่ง่าย ไม่ซับซ้อน แต่ไม่ได้รับความนิยมในปัจจุบัน เนื่องจากความเสียหายที่เกิดขึ้นกับบัสเพียงจุดเดียวก็จะส่งผลให้ทุกอุปกรณ์ไม่สามารถสื่อสารถึงกันได้เลย
	- **Ring Topology**
		![enter image description here](https://cdn.discordapp.com/attachments/974043170037956649/1032333493595811983/unknown.png)
		- สัญญาณข้อมูลจะส่งอยู่ในวงแหวนไปในทิศทางเดียวกันจนถึงผู้รับ หากข้อมูลที่ส่งเป็นของสถานีใด สถานีนั้นก็รับไว้ ถ้าไม่ใช่ก็ส่งต่อไป ซึ่งระบบเครือข่ายแบบวงแหวนนี้ สามารถรองรับจำนวนสถานีได้เป็นจำนวนมาก มีความเร็วสูงกว่าโทโปโลยีแบบบัส โดยส่งข้อมูลไปทิศทางเดียวกัน ข้อมูลไม่ชนกัน หากมีจุดใดขัดข้อง ก็จะทำให้ล่มทั้งระบบ
	- **Star Topology**
		![enter image description here](https://cdn.discordapp.com/attachments/974043170037956649/1032333562118160474/unknown.png)
		- มี ฮับ หรือ สวิตซ์ เป็นศูนย์กลาง ส่วนเครื่องจักรในแต่ละเครื่องก็เชื่อมต่อเข้าที่ศูนย์กลาง และส่งข้อมูลจากเครื่องตัวเองไปยังฮับ เพื่อให้ฮับส่งข้อมูลไปยังเครื่องจักรปลายทางอีกต่อหนึ่ง แต่ละเครื่องจะไม่เชื่อมต่อกันเอง จะรับคำสั่งจากฮับโดยตรงเท่านั้น ถ้าสถานีใดเสีย หรือสายเชื่อมต่อระหว่างฮับ/สวิตซ์กับสถานีใดชำรุด ก็จะไม่กระทบกับการเชื่อมต่อของสถานีอื่น
	- **Mesh Topology**
		![enter image description here](https://cdn.discordapp.com/attachments/974043170037956649/1032333661611229184/unknown.png)
		- เป็นรูปแบบของการเชื่อมต่อที่มีความนิยมมากและมีประสิทธิภาพสูงเนื่องจากถ้ามีเส้นทางของการเชื่อมต่อคู่ใดคู่หนึ่งขาดจากกัน การติดต่อสื่อสารระหว่างคู่นั้นยังสามารถติดต่อได้โดยอุปกรณ์จัดเส้นทาง (router) จะทำการเชื่อมต่อเส้นทางใหม่ไปยังจุดหมายปลายทางอัตโนมัติ การเชื่อมต่อแบบนี้มักนิยมสร้างบนเครือข่ายแบบไร้สาย

### Computer Network Types
 - **LAN** (Local Area Network)
	 - เครือข่ายแลน หรือเครือข่ายท้องถิ่น เป็นเครือข่ายขนาดเล็ก ใช้กันอยู่ในบริเวณไม่กว้าง ซึ่งเชื่อมโยงคอมพิวเตอร์และอุปกรณ์สื่อสาร ที่อยู่ในท้องที่บริเวณเดียวกันเข้าด้วยกัน เช่น ภายในอาคาร หรือภายในองค์การที่มีระยะทางไม่ไกลมากนัก มีตั้งแต่เครือข่ายขนาดเล็กที่เชือมโยงคอมพิวเตอร์ตั้งแต่สองเครื่องขึ้นไป ภายในห้องเดียวกัน จนถึงเชื่อมโยงระหว่างห้อง หรือองค์การขนาดใหญ่ เช่น มหาวิทยาลัย มีการวางเครือข่ายที่เชื่อมโยงระหว่างอาคารภายในมหาวิทยาลัย
 - **MAN** (Metropolitan Area Network)
	 - เป็นเครือข่ายที่ใช้ภายในเมือง หรือภายในจังหวัด เป็นระบบที่มีขนาดกลางอยู่ระหว่าง เครือข่ายแลน กับ เครือข่าย แวน
 - **WAN** (Wide Area Network)
	 - เป็นเครือข่ายคอมพิวเตอร์ที่เชื่อมโยงระบบคอมพิวเตอร์ในระยะห่างไกล เป็นเครือข่ายขนาดใหญ่ มีการติดต่อต่อสื่อสารกัน ในบริเวณกว้าง เช่น เชื่อมโยงระหว่างจังหวัด ระหว่างประเทศ การสร้างเครือข่ายระยะไกลจึงต้องอาศัยระบบบริการข่ายสายสาธารณะ เช่น ใช้สายวงจรเช่าจากองค์การโทรศัพท์แห่งประเทศไทย หรือ การสื่อสารแห่งประเทศไทย ใช้วงจรสื่อสารผ่านดาวเทียม ใช้วงจรสื่อสารเฉพาะกิจที่มีให้บริการแบบสาธารณะ เป็นเครือข่ายที่ใช้กับองค์การที่มีสาขาห่างไกล และต้องการเชื่อมสาขา เหล่านั้นเข้าด้วยกัน เช่น ธนาคาร มีสาขาทั่วประเทศ มีบริการรับฝากและถอนเงินผ่านตู้เอทีเอ็ม หรืออาจเป็นการเชื่อมต่อข้ามประเทศ ข้ามทวีปเลยก็ได้ ดังนั้น Internet จึงเป็นเครือข่าย WAN ที่ใหญ่ที่สุดเพราะเป็นการเชื่อมต่อของคนทั้งโลก

### Digital Literacy
 - **Use** การใช้โปรแกรมคอมพิวเตอร์ต่าง ๆ การเข้าถึงข้อมูลและความรู้ใน Internet การใช้อุปกรณ์ดิจิทัลต่าง ๆ
  
 - **Understand** การเข้าใจบริบทหรือเอาตัวรอดในสื่อดิจิทัลได้ เช่น การประเมินสื่อได้
    
 - **Create** การทำอรรถประโยชน์ทางสังคม การตัดสินใจว่าจะผลิตสื่อเนื้อหายังไงให้เหมาะสม
    
 - **Access** การเข้าถึง Technology

### Digital Safety & Security

 -	**Information Privacy**
	 -	**Cookie** เป็นไฟล์ข้อมูลขนาดเล็กที่ได้จากการเชื่อมต่อระหว่างเว็บไซต์และคอมพิวเตอร์  โดยคุกกี้มีหน้าที่จดจำข้อมูลที่เป็นประโยชน์ต่อตัวเว็บไซต์ ไม่ว่าจะเป็นตำแหน่งโลเคชั่นของผู้ใช้งาน หรือข้อมูลที่ผู้ใช้งานสนใจเพื่อนำไปใช้ทำการตลาดโฆษณษแบบเจาะกลุ่มเป้าหมาย ซึ่งถือเป็นการจัดเก็บและใช้ข้อมูลส่วนบุคคลเช่นเดียวกัน ดังนั้นจึงต้องมีการขอความยินยอม (Consent) จากเจ้าของข้อมูลส่วนบุคคลก่อนเสมอ
	 -	**Social Engineering** เป็นศิลปะในการหลอกลวง ล่อหลอกผู้อื่น ใช้หลักการพื้นฐานทางจิตวิทยาให้เหยื่อเปิดเผยข้อมูล เพื่อให้ได้ผลประโยชน์ตามที่แฮกเกอร์ต้องการโดยอาศัยจุดอ่อน ความรู้เท่าไม่ถึงการณ์ ความไม่รู้ ความประมาท ซึ่งการโจมตีนี้จะได้ผลดีมากเมื่อเทียบกับการโจมตีทางไซเบอร์ลักษณะอื่น ๆ โดยเฉพาะกับคนที่ไม่มีความรู้ทางด้านไอทีหรือความปลอดภัยทางไซเบอร์
	 -	**Phishing** การสร้างสถานการณ์โดยการส่งข้อความ อีเมล หรือเว็บไซต์ปลอม เพื่อเป็นเหยื่อล่อให้ผู้ใช้งานเข้ามาติดเบ็ด และหลอกล่อผู้ใช้ให้กรอกข้อมูลส่วนตัวต่าง ๆ หรือ ส่งโปรแกรมให้ติดตั้งลงเครื่องคอมพิวเตอร์ตามที่แฮกเกอร์ต้องการ ซึ่งการทำ Phishing ที่พบเห็นบ่อยในประเทศไทยมีอยู่ 2 รูปแบบ คือ Email Phishing Web Phishing
-   **Malwares**
	- **Virus** มักจะแฝงตัวมากับโปรแกรมคอมพิวเตอร์หรือไฟล์และสามารถแพร่กระจายไปยังเครื่องอื่น ๆ ได้โดยแนบตัวเองไปกับโปรแกรมหรือไฟล์ดังกล่าว แต่ไวรัสจะทำงานก็ต่อเมื่อมีการรันโปรแกรมหรือเปิดไฟล์เท่านั้น
	- **Trojan** หลอกล่อผู้ใช้ว่าเป็นโปรแกรมที่ปลอดภัย แต่จริง ๆ แล้วจะทำให้เกิดความเสียหายเมื่อผู้ใช้หลงเชื่อนำไปติดตั้ง โดยที่ผู้ใช้ไม่รู้ตัวว่ามีโปรแกรมอื่นที่อันตรายแฝงตัวมาด้วย
	- **Adware** โฆษณาบนเว็บ สร้างความรำคาญใจ
	- **Worm** สามารถแพร่กระจายตัวเอง ไม่ต้องอาศัยตัวกลาง ไปยังคอมพิวเตอร์และอุปกรณ์เครื่องอื่น ๆ ผ่านทางระบบเครือข่าย เช่น อีเมล หรือระบบแชร์ไฟล์
	- **Ransomware** ทำการเข้ารหัสหรือล็อกไฟล์ ผู้ใช้จะไม่สามารถเปิดไฟล์หรือคอมพิวเตอร์ได้ จากนั้นก็จะส่งข้อความ “เรียกค่าไถ่” เพื่อแลกกับการถอดรหัสเพื่อกู้ข้อมูลคืนมา
	- **Spyware**  แอบดูพฤติกรรมและบันทึกการใช้งานของผู้ใช้ และอาจขโมยข้อมูลส่วนตัว เช่น บัญชีชื่อผู้ใช้งาน, รหัสผ่าน หรือข้อมูลทางการเงิน เป็นต้น พร้อมทั้งส่งข้อมูลดังกล่าวไปในเครื่องปลายทางที่ได้ระบุเอาไว้อีกด้วย
	-  **Keylogger** ซอฟต์แวร์ที่ออกแบบมาเพื่อบันทึกข้อมูลการใช้แป้นพิมพ์ ข้อมูลที่ป้อนผ่านคีย์บอร์ดจะถูกบันทึกผ่านเหยื่อที่พิมพ์ข้อความบนเว็บไซต์หรือแอปพลิเคชันที่ถูกตั้งค่าไว้ จากนั้นข้อมูลดังกล่าวจะถูกส่งกลับไปยังแฮกเกอร์
